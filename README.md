
# PyHases - Hash Generator.

### Generate different types of hash of any string.


## Usage.
### <u> linux </u>

```yaml
 apt-get update
 apt-get upgrade
 pkg install git
 pkg install python3
 git clone https://github.com/thenurhabib/pyhases
 cd pyhases
 python3 pyhases.py

```
### <u> Termux </u>

```yaml
 apt-get update
 apt-get upgrade
 apt install git
 apt install python3
 git clone https://github.com/thenurhabib/pyhases
 cd pyhases
 python pyhases.py

```
